After collecting data and tracking keystrokes, you can use this web application to flag possible cheaters. Your data never leaves your browser — all processing happens locally on your computer.

Expected CSV Format (Qualtrics Export)

Only two columns are required:

  • keystroke_log - JSON keystroke data (with qid, keypresses, pastes, copies, keystroke_order, text)
  • A participant ID column (default: ResponseId; you can select a different one after upload)

Everything else (timer columns, answer columns, etc.) is auto-detected from the data.

For Qualtrics CSVs: Row 1 = headers, Row 2 = labels, Row 3 = import IDs (used for QID mapping), Rows 4+ = data.
