Expected CSV Format (Qualtrics Export)

The CSV should contain these columns:

  • prolific_id - Participant ID
  • Duration (in seconds) - Total survey duration
  • StartDate, EndDate - Session timestamps
  • TestQ1 - Answer to Q1 (intercept interpretation)
  • TestQ2 - Answer to Q2 (slope for income)
  • TestQ3 - Answer to Q3 (slope for temperature)
  • TestTimer1_* - Timing for Q1+Q2 page
  • TestTimer2_* - Timing for Q3 page
  • keystroke_log - JSON keystroke data

Row 1 = column IDs, Row 2 = labels, Row 3 = Qualtrics import IDs, Rows 4+ = data.
